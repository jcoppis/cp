Introducción a la creación de APIs

Descripción

En este curso utilizaremos express.js, es una libreria que nos facilitará varias cosas.

Requerimientos

Para la creación de la APIs necesitaremos

  1. Instalar Visual Studio Code
    • En Visual Studio Code instalar: Thunder Client
  2. Instalar Node.js
  3. Crear una carpeta de proyecto
  1. Ir a la carpeta del proyecto
  2. Desde alguna terminal ejecutar: 
    
									npm init
									npm install express nodemon
  3. Modificar el archivo package.json para que quede asi: 
    
									{
										"name": "servidor",
										"version": "1.0.0",
										"description": "",
										"main": "server.js",
										"scripts": {
											"test": "echo \"No test specified\" && exit 1",
											"start": "node server.js",
											"dev": "nodemon"
										},
										...
									}
  1. Crear un archivo server.js con el siguiente contenido: 
    
									const express = require('express');
									const app = express();
									const port = 4000;

									app.get('/', (req, res) => {
										res.send('Hello World!');
									});

									app.listen(port, () => {
										console.log(`Server iniciado en puerto: ${puerto}`);
									});

Programas Opcionales

Base de datos

Para las clases utilizaremos pocketbase

Es una base de datos basada en sqlite (en archivo de texto), que solo consta de un ejecutable. Brinda, autorización, autenticación, websocket (para actualización en tiempo real) y un ORM

  1. Ir a la carpeta del proyecto
  2. Desde alguna terminal ejecutar: 
    
									npm install pocketbase
  3. cuando se quiera usar solo se necesita hacer: 
    
									...
									const PocketBase = require("pocketbase");
									const host = "localhost";
									const port = "8090";
									const pb = new PocketBase(`${host}:${port}`);
									...
